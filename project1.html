Why Did I Make This Project?

This project was made to show that I know enough html, css, and javascript to design any layout that is given to me. I also wanted to incorporate Bootstrap and SASS into the project in order to keep the code organized. I found this design on a website advertising adobe XD, the new design tool from Adobe. Just by looking at the design I could see some challenges that would push me out of my comfort zone. There were three main things that I felt would challenge me. The fist one of the button with a gradient outline. The second was the gradient skill meter. Finally, the last challenge was keeping the code organized and clean. This would be a challenge since the are many different features going on in the design.

Button with Gradient Outline

A quick Google search can show you how to make a button. However, the button on the deisgn was something that I had never done before. In my opinion, a good developer can look at a probelm and break it down into tiny steps. I find that when I take probelms and put them into tiny steps it helps me overcome obstacles while coding. So I started with what I knew. I knew how to make a black button with white text. I created a _button.scss folder because I knew that the size of the button would work for the other ones that were different colors. Making resuable components in the design helped kept the code organized.
